A. 전용 설정 페이지

사이드바에 "설정 > API 키" 메뉴 추가. 전체 화면을 활용한 테이블 뷰.

  • 공간이 넓어 많은 키 목록 표시 가능
  • 설정 항목과 같은 맥락
  • 현재 사이드바 구조 변경 필요

C. 독립 페이지 (/settings/api-keys)

상단 네비게이션에 "API 키" 탭 추가. 별도 라우트로 접근.

  • URL로 직접 접근 가능
  • 가장 넓은 화면 활용
  • 네비게이션 변경 필요
