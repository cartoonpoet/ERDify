프리셋 + 날짜 직접 입력

프리셋 칩 + "직접 입력" 칩 선택 시 날짜 피커 노출. 유연하지만 UI가 약간 복잡.

공통 사항
• 키 이름은 선택 입력 (없으면 자동으로 "API Key #3" 같이 채움)
• 키 생성 직후 전체 키 값 한 번만 표시 → 복사 버튼 제공
• 이후엔 앞 16자(prefix)만 표시, 재생성으로만 새 키 발급
