  • 별도 진입점 없음
  • 설정 관련 기능 한 곳에

B. 설정 섹션 (사이드바 패널)

대시보드 내 Settings 패널을 새로 만들고 API 키 + 프로필을 인라인으로. MemberManagementPage와 같은 패턴.

장점

  • 공간 여유
  • 멤버관리와 일관된 UX

C. 슬라이드-인 드로어

아바타 메뉴에서 클릭 시 화면 오른쪽에서 드로어(서랍)가 밀려 나옴. 현재 화면 위에 오버레이.

장점

  • 컨텍스트 유지
  • 페이지 이동 없음
  • 공간 충분

단점

  • 드로어 컴포넌트 신규 제작
